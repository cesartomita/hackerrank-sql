# Weather Observation Station 18

Consider **P1(a, b)** and **P2(c, d)** to be two points on a 2D plane, where:

- `a` equals the minimum value in **LAT_N** from **STATION**
- `b` equals the minimum value in **LONG_W** from **STATION**
- `c` equals the maximum value in **LAT_N** from **STATION**
- `d` equals the maximum value in **LONG_W** from **STATION**

Query the Manhattan Distance between points **P1** and **P2** and round it to `4` decimal places.
