# HackerRank - SQL

Repositório criado com minhas soluções para os problemas de **SQL** do [***HackerRank***](https://www.hackerrank.com/domains/sql).

O objetivo deste projeto é praticar minha lógica de programação com foco em bancos de dados e desenvolvimento de *queries*, além de compartilhar as soluções que encontrei para os problemas propostos.

Os problemas foram resolvidos com a linguagem **T-SQL** (SQL Server).

> Este repositório é um projeto contínuo. Com passar do tempo, irei adicionando novos problemas e soluções por aqui conforme eu for resolvendo.
